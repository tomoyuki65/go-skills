# Go言語でAgent Skillsを試すサンプル
Go言語（Golang）およびGemini CLIとGoogle Antigravityを使ってAgent Skillsを試すサンプルです。  
  
<br>
  
## スキル用の各種ファイルの配置場所について
スキル用の各種ファイルについては、Gemini CLIの場合は「.gemini」配下、Google Antigravityの場合は「.agent」配下に配置します。  
  
<br>
  
## スキル一覧
- go-staticcheck: staticcheckを利用して静的コード解析
- go-lowercase: 対象の文字列を小文字に変換
